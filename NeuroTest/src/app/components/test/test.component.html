Test D2 Symbol-Digit Modalities Test (SDMT)
Tarea de cancelación de elementos visuales. Evalúa capacidad de concentración mediante velocidad y precisión. El objetivo es sustituir una serie de símbolos en dígitos a partir de una clave dada. La duración permite evaluar la pérdida de rendimiento en el tiempo.
